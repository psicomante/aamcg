AMCG - AMuCuGa
=====

Another Multiplayer Cuboids Game

### Information

Project developed in Unity4, Server and Client code live in the same codebase.

#### The tools we've used
* Unity4
* Visual Studio 2012
* MonoDevelop

#### You can have fun on
* MacOS
* Windows
* Android
