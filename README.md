AMCG - AMuCuGa
=====

Another Multiplayer Cuboids Game

### Information

Project developed in Unity4, Server and Client code live in the same codebase.

#### The tools we've used
* Unity 4.1.5f
* Visual Studio 2012
* MonoDevelop-Unity 2.8.2

#### You can have fun on
* MacOS
* Windows
* Android
* Web (with Unity Plugin)
